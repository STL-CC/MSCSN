# Data Preparation Guide

This repository does not include raw or processed datasets.

Please download and prepare data manually for local reproducibility.

## Required Datasets

- MIT-BIH Arrhythmia Database (MITBIH)
- Atrial Fibrillation Database (AFDB)

## Expected Local Paths in Current Code

The current loaders in `datasets.py` expect:

- MITBIH record files under:

  `../data/MITBIH/files/`

- AFDB NumPy files under:

  `../data/AFDB/`

  with at least:

  - `data.npy`
  - `labels.npy`

## Shapelet Discovery Input

The unified pipeline uses a representation HDF5 file under `data/representations/`:

- `data/representations/MITBIH_train.h5` for `--dataset MITBIH`
- `data/representations/AFDB_train.h5` for `--dataset AFDB`

Expected keys in HDF5:

- `full_representation`
- `all_dims`
- `all_labels` (optional for Stage 2 but generated by Stage 1)
- optional group `candidates` with datasets named `candidate_*`

## Automatic Generation

If `data/representations/<dataset>_train.h5` does not exist, `train.py` will automatically run Stage 1 representation training and generate this file.

You can force regeneration with:

```bash
python train.py --dataset MITBIH --run_stage1
```
